# PSO-CST Resonator Design and Analysis

## Overview

This repository contains MATLAB scripts developed for the optimisation and analysis of microwave resonator sensors.

The repository combines:

* Particle Swarm Optimisation (PSO)
* CST Studio Suite automation through MATLAB
* Resonator performance evaluation using custom cost functions
* Touchstone (`.s2p`) post-processing and uncertainty analysis

The code was developed as part of research involving resonator-based dielectric characterization and electromagnetic sensing.

### Main Optimisation Script

`main_pso_cst_optimization.m`

Implements a Particle Swarm Optimisation (PSO) workflow in MATLAB. The algorithm communicates directly with CST Studio Suite through the COM interface and iteratively searches for resonator geometries that satisfy the desired performance objectives.

The optimisation controls four geometric design variables:

* `t`
* `g`
* `D`
* `r`

For each particle in the swarm:

1. Geometry parameters are updated in CST.
2. The CST model is rebuilt.
3. Simulations are performed for two material states.
4. Resonator characteristics are extracted.
5. A cost value is calculated.
6. Personal and global best solutions are updated.

---

### Cost Function

`cost_function_resonator.m`

The cost function automatically:

* Updates CST model parameters
* Executes simulations for two material states
* Extracts the S21 response
* Identifies resonant frequencies
* Calculates resonance shift
* Calculates 3 dB bandwidth
* Exports Touchstone files

The optimisation favours resonators exhibiting:

* Deep resonances
* Large resonance shifts between material states
* Narrow 3 dB bandwidths

Designs producing resonances that are excessively separated are rejected to avoid comparing different resonant modes.

### S-Parameter Analysis Pipeline

`post_processing/resonator_s2p_analysis.m`

This script processes repeated Touchstone (`.s2p`) measurements or simulations.

Features include:

* Import of repeated `.s2p` files
* S21 extraction
* Resonant frequency identification
* Resonance magnitude extraction
* Averaging of repeated measurements
* Savitzky-Golay smoothing
* Validation-based uncertainty estimation
* Automatic figure generation
* CSV export of processed results

The uncertainty budget combines:

* Repeatability uncertainty
* Baseline uncertainty
* Validation/reference uncertainty

Validation measurements acquired before and after an experiment are used to quantify system stability and contribute to the overall uncertainty estimate.

## Notes on the Optimisation Workflow

The optimisation scripts were developed to interface directly with CST Studio Suite through MATLAB.

The CST project used during the original research is **not included** in this repository. Consequently, the optimisation scripts are provided primarily as an example implementation of a PSO-CST workflow and cannot be executed without a compatible CST model.

The repository is therefore intended to demonstrate:

* Scientific programming in MATLAB
* Electromagnetic simulation automation
* Optimisation workflow design
* Resonator analysis methods
* Experimental data processing and uncertainty analysis

rather than provide a fully reproducible optimisation environment.
